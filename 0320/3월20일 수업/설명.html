1. 객체란 ? 데이터를 구조화 하고 관리하며, 복잡한 데이터를 표현할때 유용하다 객체는 key,value 로 구성된 property의 집합 {}를 사용하여 생성, 각 프로퍼티는 쉼표(,)구분된다 키는 문자열, 심볼(symbol) 타입이어야 하고, 값은 어떤 타입이어도 상관없음 객체의 특징 1)객체는 참조 타입으로 변수에 객체를 할당하고 변수는 객체의 메모리 주소를 참조 2)객체는 동적이다 3)객체는 프로토타입을 기반으로 상속을 구현 const person ={ name :'홍길동', name:key , '홍길동':value age : 30, student:true, greet :function(){ key:greet 값:함수 console.log('안녕하세요') } } 2. 객체의 프로퍼티 접근 1) 점표기법 객체 이름 뒤에 점(.)을 사용하여 프로퍼티에 접근 키가 유요한 식별자 일때 person.greet() 2) 대괄호 객체 이름 뒤에 대괄호([])를 사용하여 프로퍼티 접근 키가 유효한 식별자가 아니거나, 또는 동적으로 키를 지정할때 const key = 'student' console.log(person[key]) 3. 객체의 프로퍼티 추가 수정 삭제 person.job ="개발자" //추가 person.age =31 //수정 delete.person.student//student 프로퍼티 삭제 4. 객체 반복 for...in for (let key in person) { console.log(`${key}:${person[key]}`) } 5. 객체의 종류 일반객체 {}생성된 객체 const obj ={key :'value'} 배열(array) 순서가 있는 데이터 집합 const arr =[1,2,3] 함수(function) 함수도 객체의 일종 function greet(){ console.log('안녕하세요') } 내장객체 자바스크립트에서 제공하는 객체 Date, Math, json .....
