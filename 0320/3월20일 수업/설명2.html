비구조 할당 객체 안에 있는 값을 추출해서 변수 또는 상수로 바로 선언할 수 있다. //기본 비구조 할당 const object ={a:1, b:2} const {a , b} = object console.log(a) //1 console.log(b) //2 //함수로 사용 하는 비구조 할당 function print({a,b}){ console.log(a); //1 console.log(b); //2 } print(object) Getter함수 Setter함수 Getter함수 : 특정 값을 조회 할때 설정한 함수로 연산된 값을 반환 Setter함수 : 특정값을 설정할때 마다 코드를 받아와서 실행
